# Lab 01

## Creating my first terraform deploy

Whit this lab you will create:

| Resource  | Name  |
|:----------|:----------|
| Resource Group   | demo01   |
| Virtual Network    | vnet01   |
| Subnet    | subnet01    |

## Examples

Firt of all, init the folder:
```sh
$ terrafom init
```

Optional, you can generate a plan:
```sh
$ terraform plan
```

To end this lab, apply the code:
```sh
$ terraform apply
```
